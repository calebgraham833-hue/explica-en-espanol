Consejos

  • Android: añade esta página a la pantalla de inicio y comparte texto desde Kindle/Libros → “Explica en Español”.
  • iPad/iPhone: crea un Atajo que abra esta URL con ?q= + texto copiado.
  • Esto busca definiciones monolingües en es.wiktionary.org y genera una paráfrasis simple para comprensión rápida.
Ver cómo funciona
La app usa la API pública de MediaWiki para extraer definiciones del Wikcionario en español y un simplificador local (reglas) para producir explicaciones con vocabulario más común. No requiere claves ni servicios de terceros.

Integración rápida

Android (Share Target) — Chrome puede compartir texto a una PWA que sea objetivo de compartir. Si alojas este archivo con HTTPS y manifiesto, podrás recibir texto desde cualquier app.

iOS (Atajo) — Crea un Atajo “Explica en Español”: Entrada → Obtener texto → URL https://TU_DOMINIO/explica.html?q=[[Texto]] → Abrir URL.

Modo avanzado (opcional)

Si añades una clave de API de tu LLM favorito, la app puede generar explicaciones más ricas. Pega la clave abajo (se guarda solo en tu navegador) y activa “Explicación avanzada”.
Nota: Este demo no llama a ningún servicio por defecto. Si marcas “Usar explicación avanzada”, la app intentará POST a un endpoint compatible con OpenAI en /v1/chat/completions usando tu clave. Puedes editar el código para apuntar a tu proveedor.
